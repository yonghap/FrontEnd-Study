# Numbers

JavaScript Numbers

<br>

```javascript
var pi = 3.14;
pi.toFixed(0); // 3
pi.toFixed(2); // 3.14
pi.toPrecision(2); // 3.1

isNaN(pi); // false
parseInt('3 Days'); // 3
parseFloat('3.5 Days'); // 3.5
```
