# If  (조건문)

지정된 조건을 참,거짓 여부를 판별하여 명령을 실행합니다.

<br>

## 1. If - Else

```javascript
if ((age >= 14) && (age < 19)) {}
    status = "Eligible.";
} else {
    status = "Not eligible.";
}
````

<br>

## 2. Switch

```javascript
switch (new Date().getDay()) {
    case 6:
        text = "Saturday";
        break;
    case 0:
        text = "Sunday";
        break
    default :
        text = "Whatever";
}
```
